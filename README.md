LORD VOLDEMORT SHOP
Итоговый проект интернет-магазина магических товаров по вселенной Гарри Поттера.
Развёрнутый прототип сайта доступен по ссылке: https://lisa.freedomchance.com

Состав проекта:
— предметная область;
— модель жизненного цикла;
— интерфейс;
— руководство оператора;
— тестирование;
— программная часть сайта;
— MSI-инсталлятор.

Структура сайта:
— главная страница;
— каталог товаров;
— специальные предложения;
— корзина;
— избранное;
— профиль пользователя.
Личный кабинет пользователя включает:
— мои данные;
— историю заказов;
— избранные артефакты.

Доп. информация: в рамках проектирования был разработан макет формы авторизации пользователя. Форма авторизации представлена в виде макета и подготовлена для дальнейшей реализации.
Для проекта также подготовлен MSI-инсталлятор, предназначенный для установки локальной версии сайта на компьютер пользователя.
GitHub-репозиторий содержит:
— исходный код сайта (HTML, CSS, JavaScript);
— изображения и ресурсы проекта;
— комплект проектной документации;
— MSI-инсталлятор.
